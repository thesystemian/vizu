# Légalité & Éthique du scraping — Vizu

## Ce qui est légal ✅

| Source | Légalité | Raison |
|--------|----------|--------|
| Flux RSS | ✅ 100% légal | Publié explicitement pour diffusion |
| Transcripts YouTube | ✅ Légal | Dans les ToS YT-dlp pour usage personnel/recherche |
| APIs open data | ✅ 100% légal | Données publiques, gratuites, API officielle |
| Données gouvernementales | ✅ 100% légal | Licence ouverte (Etalab, CC-BY) |

## Précautions ⚠️

- **Citer toujours les sources** dans les vidéos publiées
- **Ne pas extraire de données personnelles** (noms, emails)
- **Respecter les robots.txt** des sites scrapés
- **Rate limiting** : max 1 req/seconde sur les sites non-API
- **Vérifier avant de publier** : A2-Auditeur est là pour ça

## Principes éditoriaux Vizu

1. **Vizu ne prend pas de position politique**
2. **Vizu montre des écarts factuels**, pas des conspirations
3. **Toute donnée est sourcée** (pas d'anonymat sur les sources)
4. **Erreur = correction publique** (pinned comment ou nouvelle vidéo)

## Ce que Vizu n'est PAS

- ❌ Un média d'opinion
- ❌ Un site de désinformation
- ❌ Un agrégateur de théories du complot
- ✅ Un outil de data journalism citoyen
