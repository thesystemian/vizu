# Sources Map — Vizu A1

## Principe de tension mainstream/alternatif

Pour chaque sujet, A1 cherche **2 types de sources contradictoires** :

```
Source Mainstream (affirmation) ←→ Source Open Data (réalité)
```

## Carte des sujets × sources

| Sujet | Source Mainstream | Source Open Data | Tension potentielle |
|-------|------------------|-----------------|---------------------|
| Dépenses militaires | Reuters, OTAN.int | SIPRI Database | Objectifs vs réalité |
| Inflation | BCE, gouvernements | Eurostat, INSEE | Taux officiel vs ressenti |
| Emploi IA | Communiqués tech | BLS, Eurostat | Créations vs destructions emploi |
| Coût énergie | Communiqués énergie | Eurostat Energy | Prix annoncés vs facturés |
| IA "démocratisation" | OpenAI, Google | artificialanalysis.ai | Discours vs coûts réels API |
| Biodiversité | Gouvernements COP | WWF, IPBES | Engagements vs mesures terrain |

## Règle de sélection A1

Un sujet passe le filtre A1 si :
1. Il contient une **affirmation chiffrée** dans la source mainstream
2. Il existe une **base de données publique** pour la croiser
3. L'écart pressenti est **visuellement représentable** (bar chart, line chart, scatter)
